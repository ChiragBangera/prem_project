# Visual Templates

This folder tracks which custom visual formats are safe to reuse when answering future `q:` questions.

## Statuses

- `polished`: Approved enough to reuse automatically for social-ready output.
- `needs_review`: Useful, but needs one more design pass before becoming default.
- `draft`: Data story works, visual system is still exploratory.
- `rework`: Do not reuse until redesigned.

## Current Rule

When a question needs a visual, prefer templates from `POLISHED_VISUAL_TEMPLATES`. If the best-fit template is not polished, use it only as a draft and name the visual quality risk clearly before treating it as reusable.
